PortoLite

Creative agency and portfolio WordPress theme. This document covers setup and every option the theme adds.

1. Requirements

Item Minimum
WordPress 6.0
PHP 7.4
PHP memory limit 128 MB
max_execution_time 120 seconds, for the demo import

2. Installing

  1. Unzip the download. The theme itself is portolite.zip; the rest of the package is documentation and licences.
  2. Appearance → Themes → Add New → Upload Theme, choose portolite.zip, click Install Now.
  3. Click Activate.
  4. A notice appears listing the required plugins. Install and activate all of them before doing anything else.

If the upload fails with “the link you followed has expired”, your host's upload_max_filesize is below the zip size. Upload the unzipped folder over FTP to wp-content/themes/ instead.

3. Required plugins

Plugin Why
Advanced Custom Fields PRO The page builder. Every section is a flexible-content layout, so the theme cannot build pages without it.
Kirki Powers the Customizer controls the theme adds.
PortoLite Core Registers the Team post type, so your team members survive a theme change.

All three are offered through the activation notice. ACF PRO is commercial software and comes with the theme package rather than from wordpress.org.

4. Building a page

  1. Pages → Add New.
  2. In Page Attributes, set Template to Page (Flexible Sections). Save.
  3. A Page Sections panel appears below the editor.
  4. Click Add Row and pick a section. Fill in its fields.
  5. Add as many rows as you need; drag the handle at the left of a row to reorder.
  6. Publish, then set the page as your front page under Settings → Reading if it is the home page.

The main editor content is ignored on this template — everything comes from the sections. Use the ordinary Default template for a normal content page.

5. The sections

Section Use it for
Hero (Modern) The opening screen: headline, two buttons, a photo, a badge and trust items.
Sliding Text (Marquee) A moving band of short phrases. Pauses while the pointer is over it.
Services (Modern) A grid of icon cards, each with a link.
About (Modern) Image beside a heading, a lede, numbered points and one statistic.
Why Choose Us Image with a badge on one side, reasons on the other. Collapses to a single column if you leave the image empty.
Feature List (hover rows) A numbered list of rows; each reveals its picture when hovered or tabbed to.
Process (Modern) Numbered steps.
Pricing (Modern) Plan cards. Tick Featured on one to highlight it.
Testimonials (Modern) Quote cards with a photo, name and role.
FAQ (Modern) An accordion built on native <details>, so it works with JavaScript switched off.
CTA Band (full width) A full-width band with a photo behind it that stays still as the page scrolls.
Call To Action (Modern) A boxed call to action with two buttons.

The theme also carries the earlier section set — slider, counters, brand logos, gallery, team, contact, blog — in the same Page Sections panel.

Options that repeat across sections

  • Tone — Dark or Light. This is the section's background and the text colours that go with it; alternate them down the page.
  • Eyebrow — the small label above the heading.
  • Lede — the paragraph under the heading.
  • Icon class — a Flaticon class name, for example flaticon-check. Leave it empty and the section falls back to a built-in mark.

6. Customizer

Appearance → Customize → PortoLite Customizer holds nine sections:

  • Typography Setting — body and heading fonts, plus the dark-mode switch.
  • Header Top Setting — the bar above the header: phone, address, email, and a switch to hide it.
  • Header Setting — logo and header layout.
  • Header Social — the social links in the header.
  • Breadcrumb Setting — the page-title area: background, height, and whether the trail shows.
  • Blog Setting — archive layout, sidebar position, meta.
  • Newsletter Settings — the newsletter band and its two styles.
  • Footer Settings — footer layout, columns, copyright.
  • 404 Page — the text on the not-found page.

Header, footer and newsletter style can also be set per page, in the panels below the editor. The page setting wins over the Customizer default.

7. Colours and type

Every colour, type size, space step, radius, shadow and duration the theme uses is declared once, in assets/css/portolite-tokens.css. No section stylesheet hardcodes a value, so changing one there changes it everywhere.

Override them from a child theme rather than editing the file, so a theme update does not undo your work:

:root {
  --portolite-base: #0a7cff;        /* brand colour     */
  --portolite-accent: #ffb23f;      /* secondary accent */
  --portolite-shell: 1200px;        /* content width    */
  --portolite-section-y: 96px;      /* vertical rhythm  */
  --portolite-radius: 8px;          /* corner rounding  */
}

Fonts are Space Grotesk for headings and Inter for body text, loaded from Google Fonts. To change them, set --portolite-font and --portolite-font-two in your child theme and enqueue the families you want.

8. Dark mode

Customize → Typography Setting → Dark Mode. The switch adds a class to <body> that redefines nine tokens — background, text, headings, borders. Because sections paint with those tokens rather than with fixed colours, it reaches all of them, including any you add later in a child theme.

9. Menus and widgets

Appearance → Menus. The theme registers one location, Main Menu, used by the header and by the mobile drawer. Sub-menus work to two levels.

Appearance → Widgets has the blog sidebar and the footer columns. Leave a footer area empty and its column is not printed.

10. Blog and team

Set your posts page under Settings → Reading. Archive layout and sidebar position come from Blog Setting in the Customizer.

Team members are a post type from the PortoLite Core plugin: Team → Add New. Each carries a role, photo, contact details, biography, skills and social links. Add a Team section to a page to show them.

11. Child theme

Do your customising in a child theme. Create wp-content/themes/portolite-child/style.css:

/*
Theme Name: PortoLite Child
Template: portolite
Version: 1.0.0
*/

and functions.php:

<?php
add_action('wp_enqueue_scripts', function () {
    wp_enqueue_style(
        'portolite-child',
        get_stylesheet_directory_uri() . '/style.css',
        ['portolite-core'],
        wp_get_theme()->get('Version')
    );
});

To override a template part, copy it from the parent into the same path in the child — template-parts/sections/services-modern.php, for example.

12. What loads when

The theme reads the section list for the page being rendered and enqueues only the stylesheets and scripts those sections need. A page with no carousel loads no carousel library; a page with no scroll animation loads no animation library.

To force an asset back on — for a shortcode or a plugin injecting markup the theme cannot see — use the filter:

add_filter('portolite_asset_handles', function ($handles, $layouts) {
    $handles['css'][] = 'owl-carousel';
    $handles['js'][]  = 'owl-carousel';
    return $handles;
}, 10, 2);

13. Translating

The theme is translation ready and its text domain is portolite. Translation files go in languages/ — for example portolite-de_DE.po and its compiled .mo. Loco Translate and Poedit will both do the job.

14. Support

When reporting a problem, include your WordPress version, PHP version, the theme version, the active plugins, and what you did just before it happened. A screenshot of the page and of any error notice usually saves a round trip.
